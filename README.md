# arknights
## 本程序多为自娱自乐
### 1. 在all_in_one.xlsx的Demand页的Demand列中输入需要的材料：
负数为已拥有的数量，正数为需要的数量
### 2. 运行arknights.py
需要Python3及package: pandas, scipy
### 3. 运行成功显示Success，否则为Fail
### 4. 结果在all_in_one.xlsx的Todo页
基于需求的材料价值在Demand页的Value列：单位为理智
#### 注： 价值不考虑赤金带来的收益

数据来源于https://penguin-stats.io/
